## Stuff I've built

**GestoPilot** — Gesture-controlled presentation remote using STM32 and IMU. Built the whole thing — firmware with EMA filtering, annotation mode, and a Python config tool. The EMA dead-zone tuning took forever to get right.

**IMU Sensor Calibration** — Accelerometer and magnetometer calibration pipeline using ST's MotionAC and MotionMC libraries. Raw sensor data in, calibrated data out. Python tooling for validation.

**Digit Recognition Using Gestures** — Real-time digit recognition combining STM32 IMU sensor fusion with an ML classifier. Gesture segmentation, feature extraction, classification — all running on a microcontroller.

**Composite Diabetic Kidney Disease Indicator** — Clinical ML pipeline fusing a CNN histopathology classifier with a Random Forest biochemical model. Built a Streamlit app with Grad-CAM explainability and FAISS-powered retrieval. The multi-modal data fusion was the hard part.

**Remote-Controlled Car** — Gesture-controlled robotic car using accelerometer sensors and Arduino UNO. My first embedded project — circuit debugging, PCB soldering, and learning that hardware is unforgiving.
